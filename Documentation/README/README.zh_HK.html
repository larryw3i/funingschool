Key Features

warehousing and consuming

  1. Read food spreadsheets automatically.
  2. The simplest and most straightforward `consuming sheets`.
  3. Update sheets (warehousing, consuming, summing, etc) automatically.
  4. Reduce calculation errors.
  5. Effectively eliminate unit prices containing infinite decimals.
  6. Easy to use.

Test statistics

  1. An easy-to-use "test score entry form".
  2. Clear test results at a glance, converting table data into Intuitive images.
  4. Display comments.
  5. Effectively assist testers, especially teachers and students.

How To Use

Install Python3

Ubuntu: 

        sudo apt-get install python3 python3-pip
For `Windows`, you can install Python3 from https://www.python.org/getit/ .

Install fnschool and run it 

    # install
    pip3 install -U fnschool
    # run `warehousing and consuming` module
    fnschool-cli canteen mk_bill
    # run `test statistics` module
    fnschool-cli exam enter
Note: Read the information it prompts carefully, which is the key to using it well.
