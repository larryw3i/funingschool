funingschool

不只是一些出入库脚本。

简体中文 • English

特性 • 如何使用 • 致谢（声明） • 赞助 • 授权

特性

食材出入库

  1. 自动读取食材表单。
  2. 更简单更直观食材出库设计。
  3. 自动更新表单（入库、出库、汇总等等）。
  4. 减少计算误差。
  5. 有效地“消除”含有无限小数的单价。
  6. 易用。

测试统计

  1. 易用的“测试登记”表单。
  2. 直观展示测试结果，生成直观的图片方便发送给学生家长。
  4. 显示测试备注。
  5. 有效协助测试相关人员，尤其是老师和学生。

如何使用

安装 Python3

Ubuntu: 

        sudo apt-get install python3 python3-pip
在 Windows 上，你可以从以下链接获得二进制安装程序（请自行检验哈希值）： https://registry.npmmirror.com/binary.html?path=python/ https://mirrors.huaweicloud.com/python/ https://mirror.bjtu.edu.cn/python/

安装 和 运行 fnschool 

    # 安装
    pip3 install -U fnschool
    # 运行“出入库”模块
    fnschool-cli canteen mk_bill
    # 运行“测试”模块
    fnschool-cli exam enter
备注： 细心耐心阅读它打印的提示，这是你能使用好它的关键。
